Our Programs

We provide both undergraduate and postgraduate programs. Prospective students also can enroll in our diploma courses where they can get certified within three months of joining us

Basic

This is a 3-months program aimed to get you started in tech. Here, the basics of web/software development is thoroughly dealt with. This is more than enough to get anyone started the field. Students will be able to build a static and informative websites after this class.

Intermediate

We assume that the student already know the basics of the core fundamentals of the web (which include the HTML, CSS, and JavaScript). Therefore, we advance the student knowledge by exposing the studnets to some libraries and frameworks that can make the work easier for the students. AFter this class, the student will be able to build a fully functional website as all certified studetns will be full stack developers.

Advanced

This is a year program whihch comes with a lot of benefits. Students doing this program are expected to be very serious and ready to learn and go extra mile in solving the problems they will encounter. Students will be thoroughly taught data structures and algorithms as well as System Designs which are basic prerequisite to know in order to work for most of the big companies nowadays. This will help to improve the problem solving skills of the students
